# Handoff-archief

Sessie-handoffs voor dit project, één bestand per handoff.

**Naamconventie:** `YYYY-MM-DD-HHMM-<korte-slug>.md` — de lexicografisch laatste is altijd de meest recente.

**Bij sessiestart:** lees het nieuwste bestand in deze map (`ls docs/memory/handoffs/*.md | sort | tail -1`) om de actuele stand op te pikken. Oudere handoffs zijn historie en blijven staan.

**Nieuwe handoff maken:** `/handoff [focus van de volgende sessie]` — sla voor dit project op in deze map (projectconventie; overschrijft de default temp-dir uit de skill).
