## Gotchas (nieuw geleerd)

- **hello-pangea:** droppables nooit (un)mounten tijdens drag → actiebalk is altijd gemonteerd, verborgen via CSS (`is-actief`). Geen geneste DragDropContexts — ColumnsPanel rendert buiten de board-context.
- **Payload orderable:** herordenen via `POST /api/reorder` met `{collectionSlug, docsToMove, newKeyWillBe:'less'|'greater', orderableFieldName:'_order', target:{id,key}}`.
- **`react-hooks/purity`** verbiedt `Date.now()` in componentbody (ook server components) — servertijd als prop (`nu`) doorgeven.
- **PATCH-responses zijn depth=0** — na autosave query invalideren i.p.v. `setQueryData` met de response.
- Payload's app-header wordt op shell-views verborgen via `:has(.hm-topbar)`; op Payload-eigen schermen (editor) blijft hij staan — bedoeld.
- Playwright-MCP-artefacten (`.playwright-mcp/`) stonden in git; nu in `.gitignore`.
